# Core API

Small, direct examples for the core API surface.

- `showcase.py` runs a compact end-to-end local example with live terminal visualization.
- `drop_in_llm.py` shows `RLMFlow` used anywhere an `LLMClient` is accepted.
- `llm_query_batched.py` demonstrates batched query calls.
- `skills.py` loads user-authored skill files into the prompt.
- `view_demo.py` opens the graph viewer with synthetic snapshots.
