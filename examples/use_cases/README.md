# Use Cases

Larger examples that use RecursiveFlow for a concrete workload.

- `summarizer.py` does recursive map-reduce summarization over a long document.
- `needle_haystack.py` searches a huge in-memory `CONTEXT`.
- `needle_haystack_filesystem.py` searches many workspace files with file tools.
- `coding_agent/` is an interactive coding agent that edits files in a workspace.
- `autoresearch/` runs a research loop over benchmark-style optimization tasks.
