Project: Fast-moving rainbow boids simulation on a 2D canvas.
Requirements common to all files:
- Keep code plain HTML, CSS, and JavaScript (no build step).
- Files to produce (paths relative to project root):
  1) index.html
  2) css/styles.css
  3) js/boids.js
  4) js/main.js
- index.html must include a <canvas id="canvas"></canvas>, link to css/styles.css, and include scripts for js/boids.js then js/main.js.
- Boids must be rainbow-colored (hue cycles), fast-moving, and rendered on 2D canvas.
- Keep code modular: boids.js defines and exports or exposes a Boid class and helper functions; main.js sets up canvas, creates many boids, animates, and uses boid functions.
- Each child: write the assigned file exactly at the given path using write_file, and return only the path string when done.
- Acceptance checks (parent will verify): files exist, index.html references css and scripts, basic JS brace/paren/bracket matching, and canvas element present.
