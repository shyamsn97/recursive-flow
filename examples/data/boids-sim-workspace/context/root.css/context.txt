
Build a runnable browser-based boids simulation with plain HTML, CSS, and JavaScript.

Requirements:
- The main runnable interface is index.html at project root.
- Use separate files:
  - index.html
  - style.css
  - scripts/boids.js
  - scripts/main.js
- No build tools, no external libraries/CDNs.
- Do NOT use ES modules. Use global symbols and wire with <script src="..."></script> tags.
- index.html should contain only a full-viewport canvas (no controls/config).
- Render 100s of fast-moving, rainbow-colored boids on a 2D canvas.
- Keep it performant; a simple spatial grid is fine. Aim for 300-500 boids smoothly.
- Ensure boids are triangles oriented along velocity, with vivid rainbow hues.
- Canvas should auto-resize with devicePixelRatio scaling.
- Script order: boids.js (defines Boid, helpers) BEFORE main.js (uses Boid).
- CSS: zero margins, black background, canvas fills viewport.

Deliverables:
- For your artifact, return ONLY the file content, and put the target path on the first line as:
  PATH: <path>
